AP Create Project

Blackjack

This was the main Create Task that me and my partner Daniel made for the AP CSP Exam. It is a Blackjack game, where a user can select to yes or no to play the game, in which a random number generator is used to draw 2 random cards for you and a computer. The cards are then compared to see which one has a sum of cards closer to 21. You can also choose to draw more cards to add complexity to the game. It was really fun to make in Python, and I hope to code like this more in the future.
